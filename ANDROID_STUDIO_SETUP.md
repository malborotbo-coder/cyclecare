# تشغيل تطبيق Cycle Care على Android Studio

## الملف المضغوط
**اسم الملف**: `cycle-care-android-final.zip`
**الحجم**: 8.3 MB
**التاريخ**: 27 نوفمبر 2025

---

## خطوات التشغيل المباشر

### 1. استخراج الملف المضغوط
```bash
unzip cycle-care-android-final.zip
cd android
```

### 2. فتح المشروع في Android Studio
- افتح **Android Studio**
- اضغط **File** → **Open**
- اختر المجلد `android` (المستخرج من الملف)
- اضغط **OK**

### 3. الانتظار لانتهاء المزامنة
- Android Studio سيقوم بتنزيل المتطلبات تلقائياً
- انتظر حتى ينتهي من Gradle Sync

### 4. التشغيل
**على محاكاة (Emulator):**
- اضغط **Run** → **Run 'app'**
- اختر الجهاز المحاكى
- اضغط **OK**

**على جهاز فعلي:**
- قم بتوصيل جهازك بالكابل USB
- تأكد من تفعيل Developer Mode على جهازك
- اضغط **Run** → **Run 'app'**
- اختر جهازك من القائمة
- اضغط **OK**
