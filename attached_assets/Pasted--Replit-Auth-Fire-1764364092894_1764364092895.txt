➤ ثالثاً: المطلوب في Backend (Node/Express)
1. إزالة نظام تسجيل الدخول الخاص بـ Replit بالكامل:
   – حذف openid-client
   – حذف passport OIDC Strategy
   – حذف getOidcConfig
   – حذف أي ملفات أو routes خاصة بـ Replit Auth
   – حذف استخدام ISSUER_URL و REPL_ID

2. إضافة Firebase Auth بدلاً عنه:

import { initializeApp } from "firebase/app";
import { getAuth } from "firebase/auth";

const firebaseApp = initializeApp(firebaseConfig);
const auth = getAuth(firebaseApp);

3. تجهيز endpoint للـ logout والدخول باستخدام Firebase Session Token.
